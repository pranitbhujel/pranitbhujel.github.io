Inline Ones

$\sum_{i=0}^n i^2 = \frac{(n^2+n)(2n+1)}{6}$

To calculate the sum of squares of a number we use this $\sum_{i=0}^n i^2 = \frac{(n^2+n)(2n+1)}{6}$ formula
